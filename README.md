# graphql-voyager-docker

GraphQL Voyager in Docker

## How to use

`docker run --rm -v <your_schema_dir>:/app/schemas -p 3000:3000 -e PORT=3000 ghcr.io/rokoucha/graphql-voyager-docker:latest <your_schema_paths>`
